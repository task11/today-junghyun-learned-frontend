# HTML 참고 기능들 #

## HTML Escape ##

태그로 인식하는 문자의 이스케이프 문법

  + < : ```&lt;```
  + ```>``` : ```&gt;```
  + (띄어쓰기) : ```&nbsp;```
  + & : ```&amp;```
  + " : ```&quot;```
  + ' : ```&apos;```
  + © : ```&copy;```

#

## Aria ##

+ aria-label="~~" : 웹에 정보 전달 & Screen Reader에서 읽히게 될 내용들(따로 태그 안에 정의해서 사용)
+ aria-hidden="true" : 중요하지 않은 정보 & Screen Reader에서 읽히지 않음
