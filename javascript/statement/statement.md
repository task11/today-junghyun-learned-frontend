# 문 (Statement) #

문(statement)은 프로그램을 구성하는 기본 단위이자 최소 실행 단위이다.

  + 선언문
  + 할당문
  + 조건문
  + 반복문
##### _(문은 여러 토큰으로 구성된다.)_

**토큰, 더이상 나눌 수 없는 코드의 기본 요소 :**

**Example Code :**

```javascript

// 변수 선언문
var x;

// 할당문
x = 5;

// 함수 선언문
function foo(){}

// 조건문
if (x > 1) { console.log(x); }

//반복문
```
